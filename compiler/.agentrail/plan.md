Pascal compiler (p24p) for COR24 targeting the pv24a p-code VM. C compiler via tc24r for the compiler itself, Pascal runtime written in Pascal (dogfooded). Phase 0: expression language compiler in C (integers, booleans, assignment, if/while, writeln). Phase 1: block structure compiler in C (procedures, functions, arrays, records, var params, nested scopes, for/repeat/case, read/readln). Phase 1 runtime: rewrite runtime in Pascal (I/O formatting, standard functions, diagnostics). Pipeline: .pas → p24p (C, native) → .spc → pasm → .p24 → pv24a VM.
