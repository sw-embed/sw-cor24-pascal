Add peek/poke and memory operation routines to Pascal runtime (runtime/runtime.spc). Implement: (1) peek(addr): integer — read byte via loadb, (2) poke(addr, val) — write byte via storeb, (3) memcpy(dst, src, len) via sys MEMCPY opcode 0x70, (4) memset(dst, val, len) via sys MEMSET opcode 0x71. Also register these as known procedures in compiler/src/parser.c (existing pattern lines ~110-130). Needed for BASIC's PEEK/POKE statements and program store management. GitHub issue #5.
