Added user-defined procedure and function support to p24p. Implemented: (1) procedure declarations with value params and local vars, (2) function declarations with return type and hidden local for return value (FuncName := expr), (3) p-code emission using loada/storea for params, loadl/storel for locals, ret N for cleanup, (4) _p24p_entry trampoline as first .proc to handle PVM offset-0 execution, (5) refactored all variable access through emit_load_sym/emit_store_sym helpers. Added SYM_PARAM and SYM_LOCAL symbol kinds with scope save/restore. Two new test programs (t15_proc, t16_func) covering basic procs, functions with locals, multi-param calls, nested function calls, and Factorial. All 17 tests pass.
