Added array type support to p24p. Implemented: (1) array[low..high] of BaseType syntax in var declarations, (2) element access a[i] via addrg + (index-low)*3 + load, (3) element assignment a[i] := expr using _p24p_tmp scratch global for store stack ordering (no swap opcode), (4) non-zero-based indexing with automatic low-bound subtraction, (5) .global allocation with correct word count. TYPE_ARRAY added to type system with per-symbol array metadata (low, high, elem type, size). Two new tests: t17_array (zero-based, fill+sum), t18_array_proc (1-based arrays accessed from procedures/functions). All 19 tests pass.
