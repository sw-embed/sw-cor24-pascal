Add support for cross-unit global variable access. Units may export global variables (var declarations in the interface section). The .spi file needs to include exported globals with their types and offsets. Importing programs need to resolve cross-unit global reads/writes — either via xcall-style accessor procedures auto-generated by the compiler, or via a shared global mechanism if the p-code VM supports it. Research the best approach, implement, and test with a unit that exports a global variable read by the main program.
