Added char type and byte-level operations. Implemented: (1) TYPE_CHAR with char variable/param/return declarations, (2) char literals push ordinal value, (3) types_compatible() and is_ordinal() helpers for char/integer interop in arithmetic, comparisons, and assignments, (4) write/writeln dispatch TYPE_CHAR to _p24p_write_char (PUTC), (5) array of char with 1-byte elements using loadb/storeb and compact allocation ceil(size/3) words, (6) all arithmetic operators accept char as ordinal. ord/chr already identity functions in runtime. 20 tests pass including new t19_char.
